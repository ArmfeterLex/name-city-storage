# Name City Storage

Приложение на C# Windows Forms для сохранения и загрузки пар "Имя-Город" в текстовый файл.

## Функциональность

- Ввод имени и города
- Сохранение в файл sam3.txt (формат "Имя|Город")
- Загрузка последней записи из файла
- Отмена последнего изменения (undo) через стек
- Проверка заполнения полей

## Интерфейс

- textBoxName — имя
- textBoxCity — город
- buttonSave — сохранить
- buttonLoad — загрузить
- buttonUndo — отмена

## Куда можно встроить

- В справочные системы
- В приложения для ведения записей
- В демонстрацию работы со стеком

## Требования

- Windows
- .NET Framework / .NET Core с WinForms

## Примечание

Файл sam3.txt создаётся в папке приложения.
